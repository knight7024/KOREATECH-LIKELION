정확히 뭘 하는데요?

자체 개발된 온라인 강의 플랫폼 '클래스라이언' 을 통해 강의를 듣고 과제를 수행합니다.
교내에서는 각 학교의 운영진과 학생들이 모여 같이 학습을 진행하며
뜻을 같이하는 동료와 자신이 만들고자 하는 서비스를 직접 프로그래밍 하도록 돕습니다.
교외에서는 아이디어톤을 통해 자신의 아이디어를 자랑할 수 있고
해커톤에서 다양한 행사를 즐기며 동료들과 함께 해당 아이디어를 구현하는 시간을 가집니다.

어떤 도구들을 사용해요?

HTML + CSS + JavaScript

웹 언어들을 통해 웹 개발에 대한 기초를 다집니다.

Python + Django

파이썬을 사용하는 프레임워크인 장고를 이용하여 본격적인 웹 개발을 진행합니다.

Visual Studio Code

비주얼 스튜디오 코드를 이용하여 소스를 편집하고 작성합니다.

Git

깃을 통해 가상환경을 구축하고 프로젝트를 저장하거나 협업합니다.
